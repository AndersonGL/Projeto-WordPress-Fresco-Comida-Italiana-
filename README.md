
Fresco – Comida Italiana

Projeto desenvolvido em WordPress • Curso do Prof. Gustavo Guanabara

Este projeto consiste na criação de um site institucional para um restaurante italiano fictício, utilizando WordPress como base de estudo e prática. O objetivo é reforçar conceitos fundamentais de CMS, criação de páginas, organização de conteúdo e personalização de temas.

Sobre o Projeto

O site foi estruturado com foco em simplicidade e boa navegação. Foram aplicadas técnicas essenciais de construção de sites com WordPress, incluindo:

Instalação e configuração inicial

Criação de páginas, menus e seções

Customização de tema

Inserção de imagens e textos

Uso de plugins essenciais

Responsividade e organização visual

Funcionalidades

Página inicial apresentando o restaurante

Cardápio dividido por categorias (massas, pizzas, carnes e vinhos)

Página institucional “Sobre”

Página de Contato com formulário

Navegação leve e objetiva

Tema personalizado respeitando o padrão do WordPress

Tecnologias Utilizadas

WordPress

PHP (estrutura nativa do WP)

HTML/CSS

Plugins essenciais

Base de Estudo

Projeto desenvolvido seguindo as aulas do Professor Gustavo Guanabara, aplicando boas práticas e aprendendo a montar um site completo utilizando WordPress.
