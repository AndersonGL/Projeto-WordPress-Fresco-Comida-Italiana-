# 🥗 Fresco – Comida Italiana
**Projeto Desenvolvido em WordPress • Curso do Prof. Gustavo Guanabara**

Este repositório apresenta um site institucional fictício para um restaurante italiano, criado como parte de estudos em WordPress, abordando conceitos fundamentais de CMS, navegação, organização de conteúdo e personalização de temas.

---

## 📌 Sobre o Projeto
O site foi desenvolvido com foco em simplicidade e boa experiência de navegação. Entre as práticas aplicadas, destacam-se:

- Instalação e configuração inicial do WordPress  
- Criação de páginas, menus e seções  
- Customização de tema  
- Inserção e organização de textos e imagens  
- Utilização de plugins essenciais  
- Responsividade e estrutura visual consistente  

---

## 🍝 Funcionalidades
- **Home** apresentando o restaurante  
- **Cardápio** dividido por categorias:
  - Massas  
  - Pizzas  
  - Carnes  
  - Vinhos  
- **Página “Sobre”**  
- **Página de Contato** com formulário  
- Navegação leve, objetiva e bem organizada  
- Tema personalizado com base no padrão WordPress  

---

## 🛠 Tecnologias Utilizadas
- **WordPress**  
- **PHP** (estrutura nativa do WP)  
- **HTML/CSS**  
- Plugins essenciais
